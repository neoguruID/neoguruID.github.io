Dari Pembelajar Menjadi Pendidik

Halo! Saya Hanif Nur Fadhil. Ketertarikan saya pada teknologi sejujurnya telah tumbuh sejak saya masih seorang siswa Sekolah Dasar. Rasa penasaran terhadap komputer yang saat itu masih asing bagi saya selalu membuat saya ingin tahu lebih banyak.

Titik balik yang makin menebalkan minat saya di bidang teknologi (khususnya coding) terjadi saat SMA. Saya yang merupakan siswa baru kelas 10, diberi kepercayaan oleh guru sekaligus mentor saya untuk mengikuti OSN Bidang Komputer. Mengejutkan sekali, ternyata saya berhasil meraih Juara 1 di tingkat Kabupaten. Dari situ kepercayaan diri saya tumbuh, dan minat saya terhadap coding semakin besar.

Seiring berjalannya waktu, pengalaman saya sebagai pembelajar ternyata menumbuhkan dorongan kuat dalam diri saya untuk juga menjadi seorang pendidik. Saya ingin memantik kepercayaan diri siswa, mengasahnya, dan mengoptimalkan potensi mereka, sama seperti yang telah guru-guru saya lakukan untuk mengubah dan membentuk saya.

Visi Pembelajaran Saya

Bagi saya, belajar Informatika bukan semata-mata mencetak anak menjadi programmer. Tapi untuk melatih penalaran sistematis dan pemecahan masalah. Keterampilan ini akan berguna untuk apapun profesi mereka kelak, baik sebagai dokter, seniman, maupun wirausahawan.
